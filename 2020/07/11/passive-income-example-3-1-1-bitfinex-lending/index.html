被動收入實例 (3-1-1) Bitfinex 借貸

Bitfinex Lending 是一個虛擬貨幣中常見的借貸方式。

Bitfinex 上的借貸是用戶和用戶之間可以互相供需的市場，如果想要借錢給別人，就在 Funding book 上掛單，借出的期限為 2 ~ 30 天，收取利息的 15% 為平台手續費。需要融資 (Margin Trading) 借錢的人可以預先借款，或是在融資購買的當下由系統自動於 Funding book 上吃單。

Bitfinex 因為自己有平台上的 USD，而交易的貨幣對和主要借貸市場都是 USD，所以如果是持有 USDT 的話需要經過一次換匯成 USD，會有個交易手續費 (Maker 0.1%，Taker 0.2%)。而 USDt/USD 的價格也會些微波動，可能會有價差問題，如果短期要進出的話可能放貸的利息都會被價差和手續費吃掉。也要注意借貸的過程中，借方是不能提早解約的，只有貸方可以提早還款，所以如果借出三十天的話有機會要等到期滿才能贖回。

可以經由 Bitfinex 市場統計 看目前整體借貸市場的借貸總額以及平均利率，因為每筆交易是分開的，所以交易的成效可能會跟市場平均不同。在利率相對高點借出時，如果對方借了三十天，中途沒有提前還款，後來利率變低後融資用戶也沒有重新改借便宜資金的話，個人利率就可能高出市場平均一些。

Bitfinex 上借貸利率低的時候約為 5% 左右，高的時候可能來到 30% 甚至以上，平均的話都差不多有 10% 以上。不過如果借出後市場大幅度變化導致貸方清償失敗的話，也可能會有虧損本金的風險。如果 Bitfinex 交易所意外或惡意倒閉，像是過去的 Mt.Gox 的破產事件，那也會有本金拿不回來的風險。

因為 Bitfinex 有提供 API 可以串接，所以如果自己會寫程式的話，也可以自己串接。如果覺得麻煩，也有別人寫好的第三方開源程式 Lending Bot，像是 Mika Lending Bot、eAndrius / BitfinexLendingBot 等等。

如果不想自己寫程式，也有月租的第三方服務，像是 FULY.AI智能投資策略機器⼈、Belend 數位資產智能放貸機器人 等服務，通常會有一個 dashboard 可以看過去紀錄，可以設定一些借出策略，不過也要注意服務的安全性、穩定度以及月租費用高低。

目前撰文時，Bitfinex 上的 USD 借貸率為 7.44%，USDT 為 2.99%。
