被動收入實例 (6-1) 陽光伏特加

本篇文章將舉陽光伏特加作為一個被動收入的實際案例。

陽光伏特加 是一個太陽能板集資的平台，可以出租屋頂或是參與太陽能板出資，來賺取綠色收益。有別於傳統太陽能電廠金額較大，一般民眾無法參與投資，這類的太陽能板平台提供一個群募的管道，讓散戶投資人也可以參與太陽能板募集的過程。

以下舉近期案件 高雄港都 22 號 為範例，可以在「專案文件」中的 財務收入規劃表 看到收支預估細節。

太陽能發電的算法為：

淨收入 = 發電量 x 售電單價 - 固定支出

其中

固定支出 = 屋頂租金 + 保險 + 營運維護等等
發電量 = 日照時數 x 太陽能板功率

各地會有不同的平均年日照時數，像這個案子中太陽能板功率為 315 Wp，平均等效日照時數為 3.55 (kWh/d/kWp)，那一年就是 315 (Wp) * 3.55 (kWh / d / kWp) * 365 / 1000 = 408.16125 (kWh)。

售電單價通常為一個跟政府簽 20 年的合約躉售電價，這邊是 6.1216，那賣電收入就為 408.16125 x 6.1216 ~= 2499 元。

屋頂租金 8%，為 2499 x 0.08 ~= 200 元。

維運管理費用 2% 太陽能板售價，為 19325 x 0.02 = 386.5 ~ 387 元。

保險費用為 0.45% 太陽能板售價，為 19325 x 0.0045 = 86.9 ~ 87 元。

前十年模組回收費用為 31.5 ~ 32 元/片/年。

所以首年的收入就是 2499 - 200 - 387 - 87 - 32 = 1793 元。

之後每年衰減率為 0.7%，就可以依此類推計算。

這個例子約 12 年左右可以拿回起始投資額，之後 8 年會繼續累計收益，期初投入資金為 19325 元，期末總收入為 33257 元，內部報酬率約 6%。

投資太陽能雖然報酬並不高，但是以綠色能源的角度就是有為了這個世界的綠能貢獻一份心力。太陽能是白天發電，可以在用電尖峰的白天舒緩其他發電的發電量，如果多一點太陽能的話，白天就可以少一點火力發電，讓碳排放更少！
